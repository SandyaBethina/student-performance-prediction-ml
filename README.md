# student-performance-prediction-ml
Machine Learning and Deep Learning project for predicting student academic performance using Logistic Regression, Random Forest, XGBoost, CNN, Federated Learning, Explainable AI (SHAP &amp; LIME), and Social Network Analysis.
This project predicts student academic performance using Machine Learning and Deep Learning techniques. The system analyzes student demographic information, academic history, attendance, and engagement data to identify students at risk and help educators make early interventions.
# Features
Data preprocessing and feature engineering
Exploratory Data Analysis (EDA)
Logistic Regression
Random Forest
Linear Discriminant Analysis (LDA)
XGBoost
Convolutional Neural Network (CNN)
Federated Learning
Explainable AI using SHAP and LIME
Social Network Analysis (SNA)
Model comparison and evaluation
